# Project Vision: shrd

**URL:** shrd.co  
**Tagline:** Drop a link. Get a blog post.

## What

A universal content-to-blog engine that takes any URL (YouTube, TikTok, Instagram, podcasts, articles, tweets) and generates original, publish-ready blog posts with embedded source content.

## Why

Content is everywhere but creating original takes on that content is time-consuming. Watching a 30-minute video, reading a long article, or listening to a podcast â€” then distilling it into something shareable â€” takes hours. shrd collapses that into seconds.

**The Problem:**
- Great content exists but is locked in long-form formats
- Repurposing content manually is tedious
- Summarization tools give you summaries, not original content
- Embeds exist but nobody uses them because they require context

**The Solution:**
- Paste any URL â†’ get an original blog post with the embed
- Not just summarization â€” original title, original angle, key insights
- Ready to publish immediately

## Constraints

- **Budget:** Minimal â€” use existing tools (yt-dlp, Whisper API, web fetch)
- **Auth:** YouTube cookies already configured
- **Whisper:** Use OpenAI API (key in environment)
- **No video downloads:** Stream/extract audio only when needed
- **Storage:** Local first, optional cloud later

## Technical Preferences

### Preferred
- Node.js/TypeScript (consistent with OpenClaw ecosystem)
- CLI-first (integrate with Clawesome workflows)
- yt-dlp for video platforms
- OpenAI Whisper API for transcription
- Readability/Mozilla for article extraction

### Avoid
- Heavy frameworks for MVP
- Browser automation unless absolutely necessary
- Paid APIs beyond Whisper (for now)

## Inspirations

- **Readwise Reader** â€” Universal content capture, but we add generation
- **Descript** â€” Audio/video transcription, but we add blog output
- **Notion Web Clipper** â€” Capture anything, but we add AI processing
- **Thread Reader** â€” Unrolls Twitter, but limited to one platform

## Non-Goals (v1)

- Multi-language support (English only for MVP)
- User accounts / auth system
- Social sharing features
- Scheduled posting
- Content editing UI
- SEO optimization
- Analytics
