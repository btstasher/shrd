# shrd

> Drop a link. Get a blog post.

Universal content-to-blog engine that takes any URL (YouTube, TikTok, Instagram, podcasts, articles, tweets) and generates original, publish-ready blog posts with embedded source content.

## Installation

```bash
# Clone and install
cd ~/clawd/projects/shrd
npm install
npm run build
npm link  # Makes 'shrd' available globally
```

## Usage

```bash
# Basic usage - generates blog post from any URL
shrd https://youtube.com/watch?v=xxx

# Save to file
shrd https://paulgraham.com/superlinear.html -o ./posts/

# Extract only (no AI generation)
shrd https://example.com/article --extract-only

# Different output formats
shrd https://example.com -f html
shrd https://example.com -f json

# Different styles
shrd https://youtube.com/watch?v=xxx --style newsletter
shrd https://youtube.com/watch?v=xxx --style tweet-thread

# Custom perspective
shrd https://youtube.com/watch?v=xxx -p "ad tech executive"

# Check platform support
shrd --info https://youtube.com/watch?v=xxx
```

## Output

shrd generates:
- **Original title** (not copied from source)
- **Description** with your perspective
- **Key insights** (3-5 actionable takeaways)
- **Notable quotes** from the content
- **Embed code** for the source

## Configuration

### YouTube Cookies

For YouTube extraction, you need authenticated cookies:

1. Install a browser extension like "Get cookies.txt LOCALLY"
2. Go to youtube.com (signed in)
3. Export cookies
4. Save to `~/.config/yt-dlp/cookies.txt`

### API Keys

Set these environment variables:
- `ANTHROPIC_API_KEY` - For Claude blog generation (preferred)
- `OPENAI_API_KEY` - For GPT-4 blog generation + Whisper transcription

**Note:** Whisper transcription requires `OPENAI_API_KEY`. Cost is ~$0.006/minute of audio.
